# Elevates_labs_Task-9
Python Exception Handling and Logging
##  Objective
This program demonstrates how to handle runtime errors in Python using
try-except blocks, log errors using the logging module, save logs to a file,
and debug issues using log records.


## Features Implemented
- Use of try-except blocks
- Handling multiple exceptions
- Use of else and finally blocks
- Logging errors using logging module
- Saving logs to a file (`error.log`)
- Custom user-friendly error messages
- Simulation of runtime errors
- Debugging using log files
